# Gestion des Étudiants - Projet Java

Ce projet est un programme Java simple pour débutants qui permet de gérer les informations et les notes des étudiants.  
Il inclut la saisie sécurisée, le calcul de la moyenne et l’affichage de la mention.

## Fonctionnalités

- Ajouter un étudiant avec son **nom**, son **âge** et ses **notes**.
- Calculer automatiquement la **moyenne** des notes.
- Déterminer la **mention** selon la moyenne :
  - Excellent : ≥ 16  
  - Très bien : ≥ 14  
  - Bien : ≥ 12  
  - Passable : ≥ 10  
  - Refusé : < 10
- Menu interactif pour gérer plusieurs étudiants.
- Gestion des erreurs de saisie pour les notes et l’âge.

## Technologies

- Java SE (JDK 8 ou supérieur)
