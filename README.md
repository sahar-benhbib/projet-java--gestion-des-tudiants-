# projet-java--gestion-des-tudiants-
Saisir les informations d’un étudiant (nom, âge, notes). Calculer automatiquement la moyenne des notes. Déterminer la mention correspondante (Excellent, Très bien, Bien, Passable, Refusé). Gérer plusieurs étudiants via un menu interactif. Gérer les erreurs de saisie pour les notes et l’âge afin d’éviter les entrées invalides.
